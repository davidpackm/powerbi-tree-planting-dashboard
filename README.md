# Dashboard Power BI de suivi des projets de plantation d’arbres en Île-de-France

## Présentation

Ce projet consiste en la conception d’un dashboard Power BI permettant de suivre les opérations de plantation d’arbres en Île-de-France dans le cadre d’un **cas pédagogique**.

L’objectif est de proposer une vue d’ensemble claire des projets réalisés, de mesurer leur couverture géographique et de comparer les arrondissements à partir de plusieurs indicateurs de superficie et de plantation.

## Contexte

Les projets de plantation d’arbres génèrent des données qu’il peut être difficile de synthétiser à l’échelle d’un territoire. Un dashboard permet de centraliser les principaux indicateurs, de visualiser la répartition géographique des opérations et de comparer les zones les mieux couvertes.

Ce projet a donc été conçu comme un outil de suivi territorial permettant de :
- consolider les principaux indicateurs liés aux plantations ;
- visualiser la couverture géographique des opérations ;
- comparer les arrondissements entre eux ;
- faciliter la lecture des zones les plus couvertes.

## Objectifs du projet

Ce projet répond à plusieurs objectifs :
- fournir une vue d’ensemble des projets de plantation réalisés ;
- mesurer l’ampleur des opérations menées sur le territoire ;
- comparer les arrondissements selon leur niveau de couverture ;
- représenter visuellement la densité de plantation à l’échelle géographique.

## Structure du dashboard

Le dashboard s’articule autour d’une page de synthèse présentant les principaux indicateurs de suivi.

![Vue d'ensemble des projets](https://github.com/davidpackm/powerbi-tree-planting-dashboard/blob/382c995ccc6112dea507f24436d8ecc72c3b8549/arbres-page1.png)

### Indicateurs globaux
La vue d’ensemble met en avant :
- le **nombre d’arrondissements couverts** ;
- le **nombre d’opérations réalisées** ;
- le **nombre d’arbres plantés** ;
- la **couverture géographique totale**.

### Visualisations principales
Le dashboard comprend également plusieurs graphiques et représentations cartographiques :
- le **top 10 des arrondissements les mieux couverts en superficie** ;
- une **carte de la couverture en arbres plantés par mètre carré** pour chaque arrondissement ;
- le **nombre d’arbres plantés par arrondissement**.

## Valeur analytique

Le dashboard permet d’articuler plusieurs niveaux de lecture :
- une lecture globale de l’ampleur des projets réalisés ;
- une comparaison des arrondissements selon leur couverture ;
- une visualisation géographique de la densité de plantation ;
- une identification rapide des zones les plus fortement couvertes.

Il constitue ainsi un support pertinent pour synthétiser des données territoriales et en faciliter l’exploration visuelle.

## Outils utilisés

- **Power BI** pour la conception du dashboard et la visualisation des données ;
- **Power Query** pour la préparation et la transformation des données ;
- **DAX** pour la création des mesures et indicateurs.

## Compétences mobilisées

- préparation de données ;
- construction de KPI ;
- analyse territoriale ;
- visualisation cartographique ;
- comparaison de données entre zones géographiques ;
- data visualisation avec Power BI.

Les données utilisées dans ce projet relèvent d’un **cas pédagogique**.
